# Open Graph Image Guide

## Required Image

You need to create an Open Graph (OG) image for better social media sharing. This image appears when your portfolio is shared on platforms like Facebook, Twitter, LinkedIn, etc.

## Specifications

- **File Name**: `og-image.jpg` (or `og-image.png`)
- **Location**: `/public/og-image.jpg`
- **Dimensions**: 1200 × 630 pixels (required aspect ratio)
- **File Size**: < 1MB (ideally < 300KB for fast loading)
- **Format**: JPG or PNG

## Content Suggestions

Your OG image should include:

1. **Your Name**: "J Nagavardhan Reddy" or "Jnv Reddy"
2. **Title**: "Kotlin & Android Developer"
3. **Key Technologies**: 
   - Kotlin Multiplatform
   - Android Development
   - Cross-platform Mobile
   - Jetpack Compose
4. **Visual Elements**:
   - Clean, professional background
   - Your photo (optional)
   - Technology icons/logos
   - Brand colors (black/white/accent)

## Design Tools

You can create this image using:

1. **Figma** (Free): https://figma.com
2. **Canva** (Free): https://canva.com
3. **Photoshop** (Paid)
4. **Online OG Image Generators**:
   - https://www.opengraph.xyz/
   - https://ogimage.gallery/

## Template Resources

- **Figma Templates**: Search "Open Graph template 1200x630"
- **Canva Templates**: Search "Social Media Cover" and resize to 1200×630
- **Free Stock Photos**: https://unsplash.com

## Design Tips

1. **Text Legibility**: Use large, bold fonts (minimum 48px for main text)
2. **Safe Zone**: Keep important content within central 1000×546px area
3. **Contrast**: Ensure text is readable on the background
4. **Branding**: Use consistent colors with your portfolio
5. **Platform Preview**: Test on Facebook Debugger and Twitter Card Validator

## Testing Your OG Image

After creating and placing the image at `/public/og-image.jpg`:

1. **Facebook Debugger**: https://developers.facebook.com/tools/debug/
2. **Twitter Card Validator**: https://cards-dev.twitter.com/validator
3. **LinkedIn Post Inspector**: https://www.linkedin.com/post-inspector/
4. **Open Graph Checker**: https://opengraph.dev/
